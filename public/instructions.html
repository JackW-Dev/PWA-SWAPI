How does this all work?

Overview

We utilise SWAPI, a free, open-access API for Star Wars information!

This API powers all of our searches and is the database holding all of the information that we relay to you.

To use our system, please go to the search page and use our search form!

So, you want to know the technical stuff eh?

We leverage a free use, public API known as SWAPI to serve you with search data.

This API allows us to serve you with data about:

  • Movies
  • Characters
  • Planets
  • Species
  • Starships
  • Vehicles

The data about each of these is limited by the API response. Put simply, we can only tell you what the API tells us.

We have filtered these responses based on what we believe users would like to see, meaning that some information will be left out of the responses in some cases.

We have done this to aid with the displaying of the information and to make it easier to find the information that you are looking for.

You can find out more about the API using the documentation found here: SWAPI
